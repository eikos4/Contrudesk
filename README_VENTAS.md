## 🎯 ¿Qué es Construdesk?

**Construdesk** es una plataforma integral de gestión de proyectos de construcción diseñada para optimizar, controlar y digitalizar todas las operaciones de obras civiles.

### 🏗️ Características Principales
- **Gestión Multiempresa** - Control de múltiples empresas y proyectos
- **Perfiles Especializados** - Roles adaptados a cada necesidad
- **Control de Avances** - Seguimiento detallado del progreso
- **Gestión Documental** - Documentos técnicos y administrativos
- **Reportes de Seguridad** - Control de incidentes y prevención de riesgos
- **Sistema de Notificaciones** - Comunicación eficiente entre equipos
